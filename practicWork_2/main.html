CSS

CSS надає розробнику електронних документів набагато ширший набір можливостей роботи з текстовою інформациєю, ніж стандартний HTML

Крім способів видалення тексту:

  • підкреслення,

  • курсив,

  • жирне накреслення,

  • вибір гарнітури

  • розмір шрифта



За допомогою засобів CSS можливо змінювати такі параметри, як міжсимвольний і міжрядковий інтервал, тип регістру (малі та великі литери) і багато іншого
